## Vad jag gjorde
Ett program som hanterar användare i en lista. Man kan lägga till, söka och visa alla användare.

## 3 Alternativ jag tänkte på

### Alternativ 1: Följa mallkoden exakt
Använda mallkoden som grund och bara fylla i funktionerna. Enkelt men kanske lite tråkigt.

### Alternativ 2: Göra det som ett kundregister för Samis Jackets
Istället för "users" kan det vara kunder i min klädbutik. Mer relevant för mig.
- namn = kundnamn
- email = kontakt
- ålder = kanske VIP-status baserat på ålder?

### Alternativ 3: Lägga till extra funktioner
Typ ta bort användare, redigera användare osv. Men uppgiften säger bara 4 val...

## Vald lösning
**Alternativ 2** - Gör det som ett kundregister men kallar det fortfarande "användare" så det matchar uppgiften. Tänker på det som kunder i min butik.
