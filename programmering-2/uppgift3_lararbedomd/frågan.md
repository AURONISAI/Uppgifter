# Lärarbedömd uppgift 3 – Python

**Att lämna in:** En .py fil döpt till `lararbedomd-uppgift-3.py`

## Uppgiftens syfte
Visa att du kan:
- Skapa och använda egna klasser i Python.
- Strukturera ett objektorienterat program med flera objekt.
- Använda arv, metoder, konstruktorer och attribut.
- Testa och dokumentera din kod.

## Instruktioner
Välj ett av tre projekt:

**Alternativ 1: Bibliotekssystem** – Book, Member, Library  
**Alternativ 2: RPG-strid** – Character, Player, Enemy, Item (VALD)  
**Alternativ 3: Receptplanerare** – Ingredient, Recipe, MealPlanner

## Bedömningskriterier
- **E:** Minst två egna klasser, metoder och attribut korrekt.
- **C:** Arv, metodöverskuggning, kommenterad kod.
- **A:** Arv, flera samverkande objekt, listor och logik, UML/design, felsökning.

## Checklista
- [x] Minst två egna klasser
- [x] Objekt används tydligt
- [x] Alla funktioner testade
- [x] Koden kommenterad
- [x] Arv och flera objekt används
