## 🔢 ÄMNEN

### 1. Normalfördelning / Statistik

**68-95-99.7 REGELN (VIKTIG!):**
```
μ ± 1σ  →  68% av alla värden
μ ± 2σ  →  95% av alla värden  
μ ± 3σ  →  99.7% av alla värden
```

**MEDIAN:**
```
Median = mittersta värdet (sorterat!)

Udda antal: mittersta talet
Jämnt antal: medelvärdet av de 2 i mitten
```

**Exempel median:**
- 2, 5, 7, 8, 12 → median = 7 (3:e av 5)

**LÅDAGRAM (Box Plot):**
```
Min----[Q1----Median----Q3]----Max
        |     LÅDAN      |
        |   50% av data  |
```
- **Lådan** = mittersta 50% (Q1 till Q3)
- **Q1** = 25:e percentilen
- **Q3** = 75:e percentilen
- **IQR** = Q3 - Q1 (lådans bredd)

**Hur man använder:**
1. Titta på intervallet (t.ex. 165 till 175)
2. Räkna: hur många σ från μ?
3. Matcha med regeln

**Exempel:**
- μ = 170, σ = 5
- Intervall [165, 175]
- 165 = 170 - 5 = μ - 1σ
- 175 = 170 + 5 = μ + 1σ
- Svar: 68%

### 2. Andragradsfunktioner (Parabel)

**STANDARDFORM:**
```
f(x) = ax² + bx + c
```

**SYMMETRILINJE:**
```
x = -b / (2a)

→ Det är också x-koordinaten för extrempunkten!
```

**EXTREMPUNKT (VERTEX):**
```
1. Räkna x med formeln: x = -b / (2a)
2. Sätt in x i f(x) → får y
3. Extrempunkt = (x, y)
```

**a bestämmer parabeln:**
```
a > 0 → öppnar UPPÅT  → minimum (lägsta punkt)
a < 0 → öppnar NEDÅT  → maximum (högsta punkt)
```

**Exempel:**
```
f(x) = x² - 6x + 5   (a=1, b=-6, c=5)

Symmetrilinje: x = -(-6)/(2·1) = 6/2 = 3
y-värde: f(3) = 9 - 18 + 5 = -4
Extrempunkt: (3, -4) → minimum (a>0)
```

**Tillämpning (rörelse):**
```
h(t) = -5t² + 20t + 15   (raket)

Max höjd vid: t = -20/(2·(-5)) = 2 sekunder
Max höjd:     h(2) = -20 + 40 + 15 = 35 meter
Startvärde:   c = 15 (höjd när t=0)
```

### 3. Algebra - Förenkla uttryck

**Metod:**
1. Hitta lika termer (samma variabel)
2. Addera/subtrahera separat
3. Skriv ihop

**Exempel:**
```
3x - 2 + 4x + 5
= (3x + 4x) + (-2 + 5)
= 7x + 3
```

**OBS:** x-termer och konstanter kan INTE slås ihop!

### 4. Prioriteringsordning

**Ordning (PPPP):**
1. **P**arenteser ()
2. **P**otenser ^
3. **P**unkträkning (· och /)
4. **P**lusminus (+ och -)

**Exempel:**
```
2 + 3 · (4 - 1)
= 2 + 3 · 3      ← parentes först
= 2 + 9          ← multiplikation
= 11             ← addition sist
```
