# REGLER FÖR MATEMATIK 2 - STUDIER

## 🎯 Syfte

Denna mapp är för att:
1. **Spara frågor och svar** - Alla frågor du ställer under studier
2. **Logik och lösningar** - Hur vi löste varje problem
3. **Slutlig studieguide** - Sammanställning före prov

## 📁 Struktur

```
matematik-2/
├── REGLER.md           ← Denna fil
├── STUDIEGUIDE.md      ← SLUTLIG sammanfattning före prov
├── fragor/
│   └── session_DATUM.md  ← Varje studiesession sparas här
└── amnen/
    ├── derivata.md
    ├── integraler.md
    ├── ekvationer.md
    └── ...
```

## 📝 Hur det fungerar

1. Du ställer en fråga
2. Jag förklarar och hjälper dig svara
3. Frågan + svaret + logiken sparas automatiskt
4. Före provet → allt samlas i STUDIEGUIDE.md

## 🧠 Sparad information per fråga

- **Frågan** (originaltext)
- **Ämne** (derivata, integral, etc.)
- **Lösning** steg för steg
- **Logik** (varför vi gör så)
- **Tips** för att komma ihåg
