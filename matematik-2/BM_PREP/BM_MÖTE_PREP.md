### Uppgift 2 - Linjär ekvation
**Svar: x = 3**

```
5x − 7 = 8

5x = 15

x = 3

Kontroll: 5·3 − 7 = 15 − 7 = 8 ✓
```

### Uppgift 3 - Andragradsfunktion
**f(x) = x² − 6x + 5**

**Svar: Symmetrilinje x = 3, Extrempunkt (3, −4)**

```
a = 1, b = −6, c = 5

Symmetrilinje:  x = −b / (2a) = −(−6) / (2·1) = 6/2 = 3

Extrempunkt:
f(3) = 3² − 6·3 + 5 = 9 − 18 + 5 = −4
Extrempunkt: (3, −4)

Minimum eftersom a = 1 > 0 (parabeln öppnar uppåt, U-form)
```

**Om läraren frågar:** "Hur vet du att det är ett minimum?"
→ Koefficienten a = 1 är positiv, det betyder att parabeln öppnar uppåt som ett U, och extrempunkten är längst ner = minimum.

### Uppgift 4 - Raket (andragradsfunktion i verkligheten)
**h(t) = −5t² + 20t + 15**

**Svar: Högst efter 2 sekunder, på 35 meters höjd**

```
a = −5, b = 20, c = 15

Tid för maxhöjd:  t = −b / (2a) = −20 / (2·(−5)) = −20 / (−10) = 2 sekunder

Maxhöjd:
h(2) = −5·2² + 20·2 + 15
     = −5·4 + 40 + 15
     = −20 + 40 + 15
     = 35 meter

c = 15 → raketen startade 15 meter ovanför marken (t = 0)
```

**Om läraren frågar:** "Vad betyder −5t² i formeln?"
→ Det negativa tecknet gör att parabeln öppnar nedåt (∩-form), vilket är rimligt för en raket som går upp och sedan faller.

**Om läraren frågar:** "Vad betyder c = 15?"
→ h(0) = 15, det vill säga när t = 0 (startögonblicket) befinner sig raketen 15 meter ovanför marken.
