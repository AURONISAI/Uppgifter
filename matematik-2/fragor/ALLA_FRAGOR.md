### Fråga 1: Normalfördelning - Intervall

**Datum:** 2026-02-17  
**Ämne:** Statistik / Normalfördelning

**Frågan:**
> Medelvärde μ = 170, standardavvikelse σ = 5. Hur stor andel är mellan 165 och 175 cm?

**Svarsalternativ:**
- A. Cirka 68 % ✅ RÄTT
- B. Cirka 50 %
- C. Cirka 95 %

**Lösning:**
1. Räkna hur många σ från μ:
   - 165 = 170 - 5 = μ - 1σ
   - 175 = 170 + 5 = μ + 1σ
2. Intervallet är ±1σ → 68%

**LOGIK ATT KOMMA IHÅG:**
```
68-95-99.7 REGELN:
±1σ = 68%
±2σ = 95%
±3σ = 99.7%
```

**Tips:** Om siffrorna skiljer sig med exakt σ från μ, använd regeln direkt!

### Fråga 2: Förenkla uttryck

**Datum:** 2026-02-17  
**Ämne:** Algebra

**Frågan:**
> Vilket uttryck är korrekt förenklat: 3x − 2 + 4x + 5

**Lösning:**
1. Gruppera x-termer: 3x + 4x = 7x
2. Gruppera konstanter: -2 + 5 = 3
3. Sätt ihop: **7x + 3**

**LOGIK ATT KOMMA IHÅG:**
```
Förenkla uttryck:
1. Hitta LIKA TERMER (samma variabel eller bara siffror)
2. Addera/subtrahera dem separat
3. Skriv ihop svaret
```

**Tips:** x-termer och konstanter är OLIKA termer - kan inte slås ihop!

### Fråga 3: Prioriteringsordning

**Datum:** 2026-02-17  
**Ämne:** Aritmetik

**Frågan:**
> Vilket blir resultatet av 2 + 3 · (4 − 1)

**Lösning:**
1. Parentes: (4 - 1) = 3
2. Nu: 2 + 3 · 3
3. Multiplikation: 3 · 3 = 9
4. Addition: 2 + 9 = **11**

**LOGIK ATT KOMMA IHÅG:**
```
PRIORITERINGSORDNING:
1. Parentes ()
2. Potens ^
3. Multiplikation · och Division /
4. Addition + och Subtraktion -
```

**Tips:** "Parenteser, Potenser, Punkträkning, Plusminus" (PPPP)

### Fråga 4: Kvadratisk ekvation (enkel)

**Datum:** 2026-02-17  
**Ämne:** Ekvationer

**Frågan:**
> Vilka lösningar har ekvationen x² = 64?

**Lösning:**
1. x² = 64
2. Ta roten: x = ±√64
3. x = **8** eller x = **-8**

**LOGIK ATT KOMMA IHÅG:**
```
x² = tal
↓
x = ±√tal

ALLTID TVÅ LÖSNINGAR!
(positiv OCH negativ)
```

**Tips:** Glöm aldrig minustecknet! (-8)² = 64 också!

### Fråga 5: Median

**Datum:** 2026-02-17  
**Ämne:** Statistik

**Frågan:**
> Vad är medianen av talen 2, 5, 7, 8, 12

**Lösning:**
1. Sortera talen (redan sorterat)
2. Räkna antal: 5 tal
3. Udda antal → mittersta talet
4. Position 3 av 5 = **7**

**LOGIK ATT KOMMA IHÅG:**
```
MEDIAN = mittersta värdet

Udda antal: Ta mittersta
Jämnt antal: Medelvärdet av de två i mitten

Position = (antal + 1) / 2
```

**Tips:** Sortera ALLTID först!

### Fråga 6: Lådagram

**Datum:** 2026-02-17  
**Ämne:** Statistik

**Frågan:**
> Vad visar lådan i ett lådagram?

**Svar:**
Lådan visar de **mittersta 50%** av alla värden (mellan Q1 och Q3)

**LOGIK ATT KOMMA IHÅG:**
```
LÅDAGRAM:

   Min----[Q1----Median----Q3]----Max
          |       LÅDAN       |
          |     50% här       |

Q1 = 25% 
Median = 50%
Q3 = 75%
```

**Tips:** Lådan = IQR = Interquartile Range = Q3 - Q1

### Fråga 7: Normalfördelning - hitta intervall

**Datum:** 2026-02-17  
**Ämne:** Statistik / Normalfördelning

**Frågan:**
> Om en klass har medelvärde 170 cm och standardavvikelse 5 cm, inom vilket intervall ligger cirka 68% av längderna?

**Lösning:**
1. 68% = μ ± 1σ (enligt regeln)
2. Undre: 170 - 5 = 165
3. Övre: 170 + 5 = 175
4. Svar: **165 till 175 cm**

**LOGIK ATT KOMMA IHÅG:**
```
Hitta intervall från procent:

68%  → μ ± 1σ  → [170-5, 170+5] = [165, 175]
95%  → μ ± 2σ  → [170-10, 170+10] = [160, 180]
99.7% → μ ± 3σ → [170-15, 170+15] = [155, 185]
```

**Tips:** Samma regel åt båda håll - från intervall till % ELLER från % till intervall!
