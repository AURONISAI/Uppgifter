# Reflektion - Del 2

Sami

Det jag tar med mig fran uppgiften:

- HTML-felen var mest tillganglighet: bilder utan alt, input utan label, dalig titel.
  Lattfixat om man bara kommer ihag det.
- CSS-felen var fargkontrast och 9px text. Sant ser man direkt om man oppnar sidan
  istallet for att bara stirra pa koden.
- Det allvarligaste felet var i JavaScript: en saknad `}` -> SyntaxError -> hela
  scriptet korde inte alls. Det forsta jag gor nu nar nat inte funkar ar att
  oppna konsolen.

Insikt: en sida kan se ut att fungera men anda vara trasig pa flera plan
- tillganglighet, lasbarhet, JS - utan att det syns vid forsta blicken.
Validering (W3C, lint) hjalper att fanga sant innan det gar live.
