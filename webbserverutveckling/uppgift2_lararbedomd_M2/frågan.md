## Del 1 – Webbplats

Skapa en webbplats med åtta webbsidor där varje sida demonstrerar en viss teknik. Använd HTML, CSS och eventuellt JavaScript beroende på nivån. Webbplatsen ska ha en navigeringsmeny så att användaren kan växla mellan sidorna.

### Nivå 3 (det jag gör)
Åtta HTML-sidor, meny som alltid syns och är stylad med CSS.

1. Bild med `<img>` + alt-text
2. Ljud med `<audio>` (start/stopp)
3. Video med `<video controls>`
4. Canvas 2D + JS-rektangel
5. CSS background-image + vit text med text-shadow
6. Bakgrundsvideo som täcker skärmen (`object-fit: cover`)
7. Three.js: roterande kub med bildtextur
8. A-Frame: roterande vit kub, röd sfär, grön plan, blå himmel, namn-text

### Bonus
- Mörkt/ljust tema (toggle)
- Egen animerad CSS-bakgrund

Inlämning: mapp `mohammad-sami/`, zippad till `mohammad-sami.zip`.

## Del 2 – Identifiera och dokumentera fel i en webbsida

Källkoden ligger på Pauls repo: M2-larabedomd-uppg/del-2 (`index.html`, `style.css`, `script.js`).

Steg:
1. Identifiera felen
2. Dokumentera och analysera (vad, varför, hur jag fixar)
3. Genomför fixarna i koden (med kommentarer som förklarar)
4. Validera (W3C HTML, W3C CSS, JS-lint)
5. Reflektion
