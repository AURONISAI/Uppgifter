# M3 v3 - Katalog

Mohammad Sami Alsharef

En produktkatalog for Samis Jackets med riktiga produktfoton.

Sex modeller, 21 fargvarianter. Man kan filtrera pa farg, sortera
pa pris/namn, klicka pa en jacka for detaljer och spara favoriter
i en onskelista.

## Filer
- index.html - sidan
- style.css - design (samma stil som M2)
- script.js - all logik
- assets/logo.png - logotyp
- assets/produkter/ - 21 jpg-foton

## Funktioner
- Katalog med 6 jackor och 21 farger
- Filter och sortering
- Modal med fargvaljare som byter bild
- Onskelista som sparas i localStorage
- Vader fran open-meteo (rekommenderar en jacka)
- Random fakta-API
- Tema-farg picker
- Morkt/ljust lage
- Toast-notiser
- Mobilanpassad

## Skillnad mot v1 och v2
- v1 har en 3D-gjord jacka i CSS
- v2 har Unsplash-bilder
- v3 har vara egna riktiga produktfoton
